对“西游-伏妖篇”的一点评价

发表于 2017-02-02 | 更新于 2017-07-29 | 分类于 生活点滴 | | 阅读次数

在大年初一“西游-伏妖篇”上映后，一直没机会去观看，地处乡下，交通不便，今天终于如愿，和同学一起去影院欣赏了这部期待已久的电影，感觉很值！



电影剧情承接“西游-降魔篇”，有很多细节都能让人回想到星爷之前的作品，但是不感乏累反感，反倒有点亲切，那些陪伴我们一点点成长，给我们带来过欢笑的电影作品，在“伏妖篇”中大多能找到影子。



影片风格任然是星爷的“无厘头”，但是恰到好处，一点不过分，不反感，笑点连连，细节处理得非常完美，不得不佩服星爷“调教”演员的功夫。演员的演技可谓精彩绝伦，非常值得一看。

细细想来，“西游”系列可谓诠释了星爷的人生，那首“一生所爱”也让人感受到星爷的内心。至尊宝与紫霞仙子在城楼上相拥而抱，望着悟空离去的背影，一句“那个人好奇怪，像只狗”，很风趣，却辛酸。

影片在剧情上有两点小瑕疵：

  1. 唐僧师徒因殴打国王离开“比丘国”，接着唐僧和悟空一番对白论辩，之后师傅四人（三人+一条鱼）返回宫殿，剧情才得以继续发展。仿佛刻意安排，如果师徒就此离开宫殿，就会没了下文，这个过渡有点生硬，也没有合理的解释说明他们为何回去。或许是因为唐僧说的“不能砸了招牌”，但不够突出。
  2. 最后三徒弟和boss开打，红孩儿率先攻击，没几招就败下阵来，明显被弱化了，可是在皇宫内的战斗中，红孩儿的实力还是很牛掰的（可能是因为：放小弟给主人公升级的boss，不是好boss吧~）。对决中，红孩儿的匆匆过渡显得有点唐突，看后细想感觉他是不是来跑龙套的？

但是总体说来，影片任然值得一看，很精彩，不愧为“星爷+徐老爷”，“没有特效的特效”做的很棒，悟空真的是徐克养的那只猴子吗？

关于彩蛋，电影剧情一结束就关了，没有放到最后，不清楚到底有没有传说中的“不是彩蛋的彩蛋”。
