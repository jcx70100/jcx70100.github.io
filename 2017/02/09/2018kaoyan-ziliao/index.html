2018考研视频资料获取渠道总结（更新中）

发表于 2017-02-09 | 更新于 2017-04-18 | 分类于 考研 | | 阅读次数
文章更新于2月10日

考研复习过程中，结合一些名师的讲解视频以及相关书籍，会使自己的水平得到较快的提升，现在根据手上掌握的一些资料，将一些靠谱的考研视频资料获取方式进行一下总结

考研资料QQ群

  1. 18跟谁学考研群：361142667
  2. 18助力考研全程指导群：454840934（该群现在收费了，3元进群，先试试另外两个）
  3. 18考研资料分享群：233785385
  4. 2018考研信息分享群：291169699（进群后不要做任务，每天群主会发4个百度云链接，点一个就好，会自动加入百度云群组，获得视频文件夹）

这四个群我已经加了，也已经获得了视频的百度云文件夹，初期感觉还是可以的！

注意：

  • 加群的时候可能会显示群已满，按照提示加新的群即可。
  • 加群后，一般会有群公告告诉你怎么获取视频的百度云更新文件夹，按要求做就是了。
  • 需要指出的是，大部分群主都要求你将群内的分享信息转发到三个或者五个考研群，并且截图给他，此时我们可以自己创一个群，名称改成考研相关的，转发到自己的群后截图，再改一下群名称，在截图即可， 这样就不会因转发这种消息而被别人厌烦。

新浪微博

  1. 这里就介绍一个常用的：“居世远”，关注这个微博，不定期会发最新的考研视频资料，都是百度云链接。

“居世远”也有QQ和QQ群，不过今年开始收费了，后期观望一下，好的话就推荐，其实各家的视频都差不多，都是不要钱或者几块钱的“盗版”，我们的原则是花最少的钱，最大的提升自己，钱花多了，买个心理安慰罢了。

一直播平台

一直播平台是很多老师进行免费直播的地方，免费哦！推荐几个：

  1. 汤家凤老师——
    汤家凤老师是从去年年底开始做直播的，寒假一直在更新，内容包括数学竞赛辅导以及考研数学辅导，免费的，很是不错！
  2. 蒋中挺老师

在一直播app中直接搜这些老师的名字即可。

微信公众号

  1. 刘一男老师： 微信公众号 搜索”刘一男”即可，刘一男老师先前一直在公众号平台分享“8000词汇”和“18考研核心词汇”的视频讲解，也是免费，大家可以关注一下。
