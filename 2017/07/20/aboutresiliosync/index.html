用Resilio Sync同步考研资料

发表于 2017-07-20 | 更新于 2017-07-20 | 分类于 考研 | | 阅读次数
由于百度网盘对于非会员用户下载限速，并且和谐用户网盘文件，出于资料的分享安全性和稳定性，介绍一个新的分享方式。

需要的工具

Resilio Sync（电脑手机端都有）
官方网址

如果官方打不开或下载很慢，也可以直接点下面的下载对应版本：

Resilio Sync_windows32位

Resilio Sync_windows64位

Resilio Sync_macos版

Resilio Sync_安卓及IOS端–>应用商店搜Resilio就有了

这不是通过百度云分享！！这不是通过百度云分享！！这不是通过百度云分享！！这不是通过百度云分享！！这不是通过百度云分享！！这不是通过百度云分享！！

原因有二：

  1. 百度网盘非会员下载限速，200Kb／s，对于考研资源的下载非常耗时间，并且，重点是购买会员非常贵，考研的学生考虑买会员，买1个月绝对是不够的，起码买到考研结束吧，所以金钱上面会是一个负担！！
  2. 百度云的资源完全受百度掌控，他想和谐哪个就和谐哪个，记得我去年考研的时候，强化阶段文都官方就提出反对盗版行动，于是百度网盘一大堆考研视频全部被和谐，变成了“8秒短视频”，所以我们的资料在里面是不安全的。

综上，决定用Resilio Sync方式分享，不存在以上的两种情况。

使用方法

软件安装完成后，点击软件界面左上角的“加号”，点击“输入密钥或链接”，然后讲上面对应课程的密钥输入进去，选择文件下载路径即可进行同步。

说明

  1. 资源的更新请点击博客置顶文章或者点击此处

  2. 软件同步过程中完全自动，速度取决于自己的网速。这是个软件，不是网盘， 文件是直接同步到本地硬盘 ，所以要求大家的磁盘空间留多一点。
