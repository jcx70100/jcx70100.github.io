发表于 2017-06-28 | 更新于 2017-06-28 | 分类于 考研 | | 阅读次数
本人参加了2017年的考研，专业课是856电子线路，在复习期间购买了一写真题资料，现将资料进行整理，做成pdf文件共享，方便大家直接打印下来做题复习，省的花钱去买，真题是不便宜的！

资料整理自本人购买的真题材料还有网络，以及2017参加856电子线路考研的同学。

点击相应标题即可下载！

夏令营资料

中科院各研究所每年都会有夏令营，招收学生进行研究室的介绍，导师交流等，期间会拿到相应的资料，里面包括考研真题还有复习模拟题，很有参考价值。
这份资料是网上搜集整理得到的，没有答案，部分题目来自真题。
