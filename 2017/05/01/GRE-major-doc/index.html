考研专业课真题搜寻方法总结

发表于 2017-05-01 | 更新于 2017-05-06 | 分类于 考研 | | 阅读次数
考研中，除了三门公共课（有的专业是两门），还有一门是专业课，相对于公共课来说，专业课资料的查找就有点难度了，在此总结一下我自己考研过程中寻找专业课资料的经验吧。

对于考研专业课，首先需要明确自己考取的目标院校，因为每个学校的专业课试题是不一样的，很多都是 自主命题，所以，必须先明确考哪个学校。

接着，去对应学校的官网，在 研究生招生板块，找到“招生简章”或者“招生专业”的表格或介绍，找到自己对应的专业，了解自己要考哪些专业课。

一般来说，官网还会提供对应专业课的参考教材，这个很重要，专业课的复习一定要围绕那些参考教材，很可能考研的题目是从参考教材的课后习题中抽取。

接下来，确定了学校、专业课，就要一本正经的找资料了，大致渠道和步骤如下：

  1. 优选官网，很多学校的官网会提供历年真题，即使不是历年，也会提供那么两三份以供学生参考。

    需要注意的是，有的学校不提供真题，很多学校都“封题”了！ 对于这点，大家不要恐惧，好好复习，参考教材的课后习题认真做，“封题”很大程度上说明考题基本不怎么变！打好基础最重要。

  2. 学长学姐，如果你考的学校，之前已经有学长学姐考取过，那就再好不过了。可以联系他们，向他们请教一些专业课复习经验，还有专业课的资料，一般他们那边肯定是有的。另外，考研复习过程中的注意事项，他们也都会和你说。

  3. 对应院校的考研QQ群。可以在论坛或者直接在qq中查找对应院校的考研群，一般群文件里面会有很多真题，而且很全。

    之前准备东南大学的时候，就找了一个群，里面的专业课资料很全，比网上的全多了。虽然最后考的不是东南。

  4. 谷歌、百度、小木虫、贴吧……到处找资料，此时就要充分发挥自己找资料和整理资料的能力了，如果前面几条都没有条件满足，那就只好开启“地毯式”搜索，去 各大论坛、对应院校贴吧 等地方找资料，在贴吧可以结实一些研友，如果能力不错，还是可以找到一些资料的。

  5. 最后，终极办法——毛爷爷，去某宝，花钱买资料，或者贴吧、考研群有人卖真题，这是最后的办法了！

    注意：
    1. 对于封题的学校，就老老实实做课后习题，打好基础，如果院校已经封题，那么网上卖的很可能是假资料。
    2. 不是花钱就能买到真题，很多卖家是骗子，随便拿点卷子还有所谓的笔记来忽悠你，你也不知道是真是假，所以，万不得已别去买，除非来源比较正规，而且某宝里面很多评论是刷出来的，操作方法就是：好评+拍照，才给你返现（或者返运费-通常大十几），等你发现资料有问题，再去给差评，已经没法改了。
总的来说，考研资料的查找尽量走1-4条，按顺序一个一个试，可以全部尝试，即使前面几条已经满足，比如1-3我已经联系好了，那么4-5我也可以尝试一下。但是，凡事牵扯到钱的，一定多留个心眼，考研真题资料价格不便宜，别让骗子的恶行损害了我们的良知。
