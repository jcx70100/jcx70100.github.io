不忘初心，我的考研路

发表于 2017-03-06 | 更新于 2017-03-06 | 分类于 考研 | | 阅读次数
记录我从准备考研到25号考结束的一些心路历程，还有心灵感悟；记录我的内心成长过程，以及世界观的变化。对考研所用的资料进行了一些评价，总结了对我有用的资料还有一些经验。

不忘初心，继续前进

我的考研路

时间很快，已经考研结束了，当最后一门课考试结束的铃声响起，我难以想象，就这么结束了，没有什么仪式，没有什么喝彩，甚至连喜悦都感受不到，平平淡淡。两天的考试，没什么紧张情绪，感觉就是忽忽悠悠就进去了，然后就结束了，考得怎么样还不知道，也不想对答案，总体应该还是可以的。

现在，趁着考研的状态还在，赶紧写下我的考研历程，给学弟学妹参考，防止到了明年六月，在表彰大会上交流经验，都是干巴巴的话语，那种奋斗的精神已经不在，也没什么真正的参考价值。

首先说说我为什么考研。很简单，最初的目的是为了找个好工作，毕竟当今社会还是注重物质，没了物质基础，很难追求高雅生活。我考研复习最注重数学，相信很多人都是如此，理工科学生数学不好还谈什么研究？14年买了李永乐的复习全书，开始复习，但是不系统，中间杂事特别多，断断续续，效果不好。真正意义上开始复习，是从6月份开始，全心全意，天天泡图书馆，不是在宿舍，就是在图书馆，或者在从宿舍去图书馆的路上。这个过程中，接触了很多老师，很多新闻，渐渐觉得，考研不是为了找多么好的工作，而是为了去接触更优秀的人，提升自己的精神境界和培养一种学术精神。有时候想想，我们这一辈子，可以通过短短将近一年的努力来改变自己的命运，这样的机遇除了高考，恐怕也只有考研了。所以，不管我们基础怎么样，平时表现怎么样，还是很推荐大家考一考的，不逼自己一把，怎么知道自己是那么强大？只要努力，就充满了可能。

陪伴我们经历整个考研历程的，是那么多的书，有的很厚，有的很难，有的黑白压抑，有的色彩纷呈。很多人都有这样的习惯，或者说是癖好，就是凡是觉得好的，别人推荐的书，我就要买，甚至看到别人买了自己也买，就是别人有的自己也要有，买回来网书架上一放，一排排整整齐齐崭新靓丽，那是没有用的。不是说买了书，就等于掌握了知识。今年的英语一大作文就很好，标题“有书与读书”，买了书放在书架上睡觉等于不买，花了那么多钱，一定要发挥它的价值，考研图书都是不太便宜的！考研书籍不求多，一本就好，最多两本，不在多，在于啃透。我一开始就走了这样的弯路，数学买了《张宇36讲》（3本，贵啊！）、买了《复习全书》（加上配套的真题解析，基础过关660，分阶练习）、《李永乐线性代数辅导讲义》，发现一本都没看完！张宇的压根没看，重点看了复习全书。现在想来，结合自己的经验，写一点数学上的建议：数学一定要多做题，多做题，多做题！我从6月份开始就跟着汤家凤老师学的，看视频，记笔记，之前买的书翻都没翻！听完课就复习笔记，做习题。推荐《接力题典1800》，是汤家凤老师出的，里面题目绝对不止1800道，每天就是不断练题目。一开始跟视频复习，看完视频复习笔记做对应的习题，等基础班结束，1800基础篇应该结束了（基础篇的题目并不基础），然后开始回顾错题，反复的做！从基础班结束到暑期强化开始之前这段时间，就是不断练习题目，要有耐心，别以为自己会了，每一道题都要动笔写，而且是完整的写，从第一个“解”字开始，写到结束，因为很多题目我们看似会了，真正到了考试，会发现同样的题目，我们写到卷子上，那个逻辑是不清晰的！！完整写的目的就在于此，练习思维的严密性，使解题过程逻辑清晰。到了暑假，8月份中旬，开始看汤老师的强化班，依然是记笔记，复习笔记，做1800强化部分，不断重复，学习不就是重复吗？强化结束后，1800啃透，基本上没有难题，此时的水平上考场没有问题，汤老师的学生就这么自信！所以，对于18的学弟学妹，我不含糊，不保留，只把有用的告诉你们，少走弯路，别浪费了钱，还有最宝贵的经历。一直到考试之前，我把1800做了4遍，准确的说是3.5遍，最后一遍时间很紧，重点在政治英语，1800的强化部分没写。今天考下来感觉良好，虽然由于粗心丢掉了满分，但是最后结果应该不差。

综合来说，数学推荐的方案就是：汤家凤视频+1800（李永乐老师的线代讲义也很不错，很推荐）。汤老师的视频我会告诉大家怎么获取，不要报班！数学上，纵观考研历程，我觉得很nice的老师就是汤家凤和李永乐。李永乐老师当之无愧的线代大王，推荐他的线代课程。汤家凤老师理论功底很厚实，讲课没废话，给你满满的正能量，不是鸡汤，而是谆谆教诲——学会吃苦、爱国、学术自信还有年青一代的担当。书籍方面就是《1800》（这是习题集）还有《李永乐线代复习讲义》，两本书50不到。当然，汤老师的《复习大全》还有《高数复习讲义》、《线代复习讲义》也很经典，条理清晰，很清爽。其他书籍不做评论，复习全书题目比较老套，36讲很呵呵。如果已经买了，也没关系，最后理论体系建立完成后，可以看看其他老师的书籍当做消遣，补充点别人的思想解法也算是锦上添花。当然，每个人的情况不同，我只说我的见解。很多老师骗了学生的钱，浪费了学生的精力，把你卖了你还说他好。

数学上想说的就这么多，资料不在多，踏踏实实一步一步走好，切记浮躁。学好数学，离不开做题，需要大量习题的调教！还有一点，很重要：我在复习的时候就发现很多同学喜欢笔记本或者书本摊着，一遍做题一边看书，遇到不会的就翻笔记，这是很不对的！这样的做法，同样的题目，第二次遇到还是不会。做题目首先要摆好“姿势”，书本笔记合上，专心做题，不会的做好标记，做完后重点看解析进行思考整理，第二天做题目之前优先把这些不会的题目进行回顾。一开始肯定磕磕绊绊，但一定要坚持，考研想成功，首先要学会坚持。又想到一点：考研视频有很多，专看一个老师的就好，在基础阶段，理论体系没有建立之前，看多个老师的讲解会走火入魔，因为每个老师的思想不同，切记贪婪，崇尚专一！没会走之前别想跑，考研从来不是一天两天的事情。

英语方面，我走了很多弯路，买了很多单词书，很多很“牛”的书，后来发现全是“骗子”。英语关键是词汇和阅读，而阅读的基础在词汇，《恋恋有词》、《10天搞定考研词汇》都买过，看过，卵用不大。或许是我不走心吧，最后我还是用的自己归纳的真题单词，感觉好过任何书籍！这个好东西会分享给大家，当初在做词汇整理的时候就做的电子版，方便传阅。期间还看过刘一男老师的词汇课，还不错，现在18的孩子准备时间充裕，推荐大家看一下，刘老师会带大家认识单词的逻辑性。上完刘老师的课，我晚上在床上翻开恋恋有词，第一句话就是：什么书？单词太没逻辑了，纯粹的把长得像的放一起而已。在这里没有贬低别的老师的意思，都是我的真心体验，不忽悠。单词书可以考虑刘一男老师的，不过我没用过，我是直接看的视频（有电子版讲义），讲的真心很好。另外，我归纳的真题词汇也会分享。单词需要天天巩固学习，不能停，直到考研前一刻。阅读这块，需要强大的词汇量支撑，必须坚持。阅读存在方法！但是不是所有方法都有效，任何老师传授给你的方法，用最新三年真题去检验！暑期过后在考研群里看到最多的就是“某某老师讲的完型的方法用不上啊！某某老师的方法好像没用，好焦急，好迷茫，感觉考不上了……！”所以，不要谁都信，他们看中的是你的钱包，催你打开钱包，考研对我们来说是追求更好未来的阶梯，对一些机构来说是重大的商机，因为现在这个社会，学生的钱是最好赚的！阅读和作文方面，我推荐何凯文老师，他的KK三步法很牛，自从学了三步法，历年真题基本全对啦！不过今年的可能做的不好，考前专心背政治，半个月左右不碰阅读，生疏了。他的作文模板也很高雅（其实不是他的，是他从名人大家的演讲词出筛选出来的语料，很有用）。昨天考英语，大作文一马平川，一气呵成，相信老师看到那些句子会眼前一亮！何凯文老师10月份会推出五夜十篇直播课，非常建议看一下，跟完五天的课程，阅读基本上没问题了，100多快吧好像，今年的视频我有，我是免费获取的资源，没花钱，但真的很有用。

综上，英语推荐：刘一男老师词汇（可以买他的词汇书，20块吧，不贵）、何凯文老师的阅读和作文（看视频+资料，网络直播课的话，会有配套讲义，我听他的作文直接用的电子版讲义）、英语黄皮书（真题集，这个必须有，不然怎么练习？）

政治方面，看蒋中挺老师的，良心教师，他的直播课基本全部免费，今年唯一的收费直播是考前的点睛班，是文都合同规定每年至少一个收费课程才选择收费方式的。他让我学到的不止是知识，更有做人的底线，作为男人的担当，作为社会的年轻力量，我们肩上的使命还有我们对于这个民族的责任。不是我被洗脑，考研让我想明白很多事情。所以，如果你很迷茫，考研吧，人需要目标，需要有理想信念，否则，你最终会去了你当初最不想去的地方！
很多人推荐原命题组肖老师，我不评论，我只说我用过的，虽然我买了他的书，但是看了三个章节就放书架上了。蒋老师的书很好，方法很独到，推荐给大家，到时候大家就会领略到这个胖胖的男人的魅力，更能感受到其明确的做人底线，和对学生的关心负责。我的政治起步比较晚，看了蒋老师的强化和冲刺班，没有系统的练习题目，也没有详细的看知识点，考前背的他的《五套卷》还有《最后一套模拟试题》，以及《真题预测百分百》的部分重点知识，昨天考试全部用上，不忽悠。说自己押中了的老师很多，但是有明确出版物作为押题证明的不多，押中题目但是答案可以得分的更不多。没错，押中题目是一回事，答案能得分才是关键。
综上，蒋老师在九月份开始出版的《冲刺考点必背》、《真题预测百分百》、《五套卷》和《最后一套模拟试题》（每本都是十几块，双十一还打折！）真的是必备的。有这些够了，就算零基础，考前背完五套卷和最后模拟试题也能过线。

到这里我要揭开一个秘密了，我说的那几位老师，均来自文都教育。汤家凤老师是我在考研中遇到的教的最好的，理论扎实，条理清爽，跟他学，你的数学会很棒，不信的话我们用最后的分数说话；刘一男老师的单词讲解是最让我推崇和接受的，我们理工科的学生，做事讲究逻辑，相信你们会喜欢刘一男老师给你介绍单词的内在逻辑；何凯文老师在阅读和作文上简直没的说，真的.

说这么多，不是为了做广告，纯粹是我的亲身经历的总结，好让18的你们少走弯路（我们走的弯路实在太多了！），趁今天刚考完，趁我还有这份心境，写点对你们有用的东西。同时，要告诫大家的是，无论是书籍还是视频，认定了哪个老师请一直坚持。不要既看这个又看那个，人的具体的主体的生命和精力是有限的，不可能事事亲身实践，我们也只好把我们的感悟写给你们作为间接经验，一定与你们的实际情况相结合。选择哪个老师，什么资料我没有权利来管，但请做一个从一而终的人，就像谈恋爱，不要脚踩两只船，考研就像你在和未来谈恋爱，一定专心，切记浮躁。如果实在想看别的老师，数学方面可以，在知识体系建立完成后，看看别的老师的资料，取其精华，去其糟粕，批判继承，反而锦上添花。

还是那句话，脚踏实地，切记浮躁！决定考研，请坚持走到最后，不要中途放弃，对自己的决定负责，对自己的未来负责，这将是今后几年甚至十几年中，人生自信的强大源泉。

最后，我考研结束了，想想过去一年，专心考研，对班级和学院做的事情屈指可数，有些惭愧。

新的篇章，等待你们去书写；祖国的未来，有你们会更加精彩；当今社会，适合大有作为的我们，没有哪个时段能比现在更能让我感受到作为一个国人的骄傲和自信，祖国在不断强大，我们的明天会更好，而这一切离不开我们共同的努力！看好你们！！
