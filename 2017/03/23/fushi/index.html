中科院一志愿“复试+选导师”的感想

发表于 2017-03-23 | 更新于 2017-04-12 | 分类于 生活点滴 | | 阅读次数
3月20号我参加了中科院上海微系统所的复试，写一点我的复试心得，还有我被怼的过程！

复试前

经过半年多的努力，考研初试取得了不错的成果，总分380，不算高，但是肯定可以进入“中科院上海微系统与信息技术研究所”的复试了。
之所以选择一个研究所，其实是走投无路。一些名校，竞争太激烈，我的本科是电子科学与技术，电路设计方向，而在这个领域比较厉害的有
复旦、清华、东南等，历年分数线也不低，而且复旦的专业课有点吓人，思前想后，报考了中科院，一来竞争小点，二来导师很多，也都很厉害，
资源多，学术环境好。现在看来，如果考那些名校，380还凑合，不过都是后话，选择了就不能后悔。

在焦急的等待后，复试通知终于在官网发布，我总分排名第八，还是有很多人比我高的。
复试的准备我不是很用心，因为觉得以我的分数，过复试没问题，而且所里面也有几个做电路设计的导师，我也比较有把握选到他们。
在个人简历的自我介绍里面，我介绍的所学课程重点列了电路相关的，比如《模电》、《数电》、《cmos集成电路设计》等等，毕竟我是要选电路方向的，
而且导师在面试提问的时候，会根据你介绍的科目进行提问，我这么写相当于一种引导吧。当然，我还写了《半导体物理》，防止导师们觉得我“太偏”，
毕竟微系统所，最厉害的是MEMS，电路设计不是专项。

复试分为专业面试还有英语听力，在复试之前，从交流群里面了解到很多电路方向的导师都不带学生了，这就使得仅剩的还带电路的老师很抢手，
所以我在复试之前还特意看了一下半导体的笔记，考虑到电路方向导师很少，那么面试我的导师大部分都是传感器、器件还有材料方向，
势必会“炮轰”我的半导体物理。

参加复试

20号复试当天，上午是体检还有学校的介绍。学生待遇很不错，不仅免学费，还会有补助和奖学金；还听了各个研究室的介绍，都挺不错，
听完感觉每个研究室都想去，都有大牛，都很厉害。

下午一点开始专业面试，微电子分两组同时进行，因为微电子考生人多。我排小组第二，有点紧张，坐在门外的长凳上，把自我介绍在心里背了一遍又一遍，
这个时候也没心思在脑子里想可能提问的专业课了，能把自我介绍流利平和的说出来就不错了。面试之前，解老师还好意的和我们说不要背英语自我介绍了，
因为面试比预计时间开始的晚，而且晚上有英语听力的考核，所以她和导师们说不要求我们进行英文介绍，这使我紧张的心情有所舒缓，不过还是有点不过瘾，
因为我英文介绍背的最溜，溜到中文介绍都忘了。

第一位同学出来后，我敲门进去，关好门，鞠躬，向老师们问好。一共五位老师负责面试，我坐在他们长桌的前面的凳子上，端正坐姿。
导师让我介绍一下我自己，我就按之前准备的开始背诵，也不是很紧张，就平常语速，偶尔也会停顿想一下，但不影响整体效果。
差不多两三分钟，我的自我介绍结束了，导师开始提问。

导师先问了我为什么报考微系统所，谈谈我对微所的看法。这个我当时回答的有点乱，尽量往学术环境，氛围，以及国家地位上面靠。
接着另一位导师指着我的成绩单，说我的本科方向比较偏向电路，我说明原因后，导师又看了看我的成绩单，说我的半导体物理成绩还不错，
接着就开始“炮轰”半导体了，和我预想的一样。第一位问我：请谈一谈直接带隙半导体和间接带隙半导体。我听完就懵了，记得笔记里面没这个玩意儿，
应该没学过，于是我摇摇头，说不是很了解，配个笑脸。这时，一开始让我自我介绍的老师问我：你知道硅吗？我脱口而出，硅是半导体材料！
这时，刚刚问我问题的老师马上说：那硅是直接带隙半导体还是间接带隙半导体？我内心一万个***，怎么老揪着这个不放啦？我随便猜了一个：直接带隙半导体！
那老师看看我，然后点一下头，在纸上写了什么。后来才知道硅是间接带隙半导体，砷化镓这类化合物半导体才是直接带隙半导体。

这时，导师又问我，知不知道迁移率？这个我当然知道，笔记本上有！他让我简单介绍一下。
我开始从半导体物理的结构框架说起，半导体物理研究材料的半导性能，即其导电性。而半导体导电性分别由载流子数量和载流子运动速率共同影响。
在描述载流子的运动速率这一块，主要就是用迁移率，通常空穴的迁移率小于电子的迁移率。

接着，问我带隙半导体的那个老师又问我：迁移率有哪些影响因素？又把我搞紧张了，我想了一会，回答说：主要是温度吧，有时候掺杂浓度也可能有影响。
（不过我觉得这个回答太准确）此导师接着问，在外加电场时，迁移率会有什么变化？这时我脑子是空白的，是好用电子在电场中的运动来类比解释，说：
要看载流子的运动方向和电场方向的关系…………（巴拉巴拉说了一堆）

之后，另一个女老师说，你可以谈一下np结吗？哎呀，这下我高兴了，虽然半导体物理面第六章pn结没讲，但是在模电还有半导体器件里面，
对pn结的理解还是比较透彻的。说到：pn结在工艺上目前分为两种，大部分还是以掺杂扩散，在同一块硅片上形成pn结为主，这种pn结的杂质浓度随着pn结的长度
是呈现缓慢变化的关系，又称缓变结；另一种是通过高温键合，这样形成的pn结成为突变结。同时，我们通常意义上的pn结实际上是动态平衡pn结，其内部载流子在不断运动，是一个动态平衡的过程……

讲的过程中，老师们都看好了我，我自喜，难道我说的还不错？

接着，让我自我介绍的老师问我：你英语怎么样？英文自我介绍准备了吗？我说：就是把中文翻译一下。他点点头，说：来过上海没？我说来过！
然后他说：那你用英语介绍一下上海！！！ 我内心一百个大写的***，早知道说没来过了！！！
于是，我憋了半天憋出三四句话，导师转向别的问题。

这时，刚刚问我pn结的老师又问我：你可以详细描述一下动态平衡pn结的形成过程吗？
我巴拉巴拉说了一堆，反正过程是清晰的，核心就是最后扩散运动和漂移运动平衡了。

最后，导师问我比较想去哪个研究室，或者有没有中意的导师，问我让我学传感器学不学？
我笑着说，有电路方向最好，没有的话，一室传感器，二室，八室超导都可以的。学传感器也可以，不过在器件这一块的要好好深入学习。

然后导师说可以结束了，我起身鞠躬，向老师们道谢，然后出门。

专业面试算是结束了，总体感觉，还可以。

然后就是在门外和其他没有面试的同学交流了，我是第二个，后面还有许多同学在等待面试。

晚上的英语听力，都是常规题型，不过听力只放一遍！！三种题型：听写（有单词有短语、20空）、短对话（15空）、长对话（15空），
但是都只读一遍，不像平常考试里面，听写会读三遍。

整个过程1小时，结束后，我们回宾馆或者回找到所，今天就算结束了，等待明天的结果。

出结果

第二天，9点出结果，我们八点多就在会议室静静等待老师。

结果出来了，老师先把每个组每个人的复试分数（初试-50%，专业面试-40%，英语听力-10%），按排名报。

先报的材料组，材料组人不多。报到小伙伴的名字时，我还高兴一下，以为报道名字就是通过的，谁知分数没过60，兴奋瞬间就化为泡影，
有点悲伤，不知道如何是好，也给我的成绩一点不安。材料组报完，报微电子组，还好，我是第三，心里松了口气。这时隐约听到小伙伴吸鼻涕的声音，
看来是哭了，我有点不知所措。尽量安慰一下，感觉还是很残酷的，今年刷的人比去年多多了，自己心里也难过，感觉很可惜。

选导师

安慰了一下小伙伴，但小伙伴示意我赶紧去找导师，别浪费时间，好好加油。

我拿上表格，上面有每个导师的方向，名额数，办公室地点。朝着心仪的几个导师那边本去。

程新红老师

第一个去的程新红老师那边，她做的大功率器件方向，电路设计方向的导师已经没有了。敲门进去，和程老师问好，说明来意。
程老师看了一下我的简历和成绩单，说到：“常州工学院？说实话，你的学校我没听过！”
我知道我的本科学校不是特别好，双非，我只好陪个笑脸，说是二本院校。

程老师：“你们本科大部分课程都是电路设计一块，而且看你的课程设计还有做过的项目，数字的偏多！我这边大部分是模拟的，数字很少。”

我解释道：“模拟和数字我们都学的，只是在实践方面，数字做的比较多。”

程老师：“mos的结构熟悉吗？”

我：“比较熟悉，主要是在衬底上面通过掺杂，构造出源极和漏极，然后我们在栅源之间加电压，当电压大于阈值电压，就会形成导电沟道，
但此时还没有电流，我们再在漏源之间加电压，此时就会有电流；同时，控制漏源电压的大小，还能影响导电沟道，使器件工作在线性区或者饱和区。”

程老师又问：“你能说一下欧姆接触和肖特基接触吗？”

说实话，有点懵，这个学过，在大功率器件这门课里讲过，但是是大四的课程，专心考研没怎么听。我笑着摇摇头，哎。

老师说：“你刚刚说的掺杂就是欧姆接触！”

我表示明白的点点头。

她看看我做过的实践经历，看到我做过全国大学生集成电路大赛，问我“你做过用于SOC的低功耗带隙基准源”，那我问你“什么是带隙？”

我又懵了，记得之前就这个问题专门查过资料，但现在就是想不起来，只好自愧的摇摇头。

她又问：“那你知道，我们通常怎么实现一个恒流源吗？”

我的第一反应是MOS的栅源加恒定电压，其特性就趋向于恒流源，不过程老师的回答是“你这是另一个问题！”

接着说：“我这边的方向，重点还是在模拟，我已经有一个学生做电路设计方向了，我还是建议你好好考虑一下！”

意思就是婉拒我了！哎，好吧，没学好，没办法。

我还是笑着和程老师道谢，礼貌的鞠躬，走出来，找下一位导师。

李凌云老师

出来后，心里还是比较难受的，被怼的太惨了，怼学校，怼基础，哎，没办法，这种滋味只好自己承受。

接着，看看名单上面，有个叫李凌云的老师，做太赫兹，是电路方向，还有一个名额。我赶紧跑去他的办公室，发现已经有一个学生在和他交流，
我礼貌的向老师问好，交上我的简历还有成绩单，他看了看，直接和我说：“不好意思，我这边已经有两个学生再等了！”

我礼貌的回答：“好的，谢谢老师！！” 他的意思就是不要我咯！哎

我接过成绩单和简历，走出门，找下一位导师。

于广辉

接下来，我就把方向指向超导这边，在老师介绍的时候，也说超导会是一个很热的方向，出国机会也很多。

第一个找的是于老师，可惜于老师不在，要下午一点才回来。和我交流的是另一个老师，说于老师现在主要是研究石墨烯材料，
和超导关系不是很大，而且我的本科偏电路方向，让我好好考虑。然后还和老师交流了读博的问题，他指出，做超导这块，必须读博，
不然出不了什么成果，对老师本人来说，也是名额的浪费。

简短交流之后，留了电话号码，说于老师回来后联系我。

任洁

下一个拜访的是任洁老师，她的方向是超导数字电路，属于超导应用，而且和数字电路有关。

见到任洁老师，简单说明来意，交上我的成绩单还有个人简历，她仔细的看了一下。

“国家奖学金是什么呀？”“你会跳水？哦，不是，是做跳水比赛志愿者。”……

问我：“你做电路仿真用的是什么软件？”

这时她的电脑屏幕上正好有Cadence的界面，我就指着电脑说：“就是这个，我用的是Cadence IC”

她又说：“你还参加过集成电路比赛，那你说一下电路设计的基本流程是什么？”

我开始遮遮唔唔，因为这个比赛毕竟没拿奖，我都坦白了。

接着，任老师说：“四大力学你学过多少？”

我说：“没太多了解”

她说：“我已经有一个负责电路设计这一块的学生了，我想找一个有器件功底的学生，你看你量子力学以及固体物理这些最基础的你都没学。”

这样吧，我把你的简历拍一下，我再看看其他同学，最后觉得你合适的话我就联系你。

就这样，我灰溜溜的出来了，有点沮丧，想不通，为什么我初试第八，复试第三，却没老师要我！我有这么差吗？

还是我高分低能？就因为我学校不好，基础也不好？好吧，看来导师更注重基础，在他们眼里只在乎你是不是对他们将来的研究工作有帮助。

接下来基本就是到处游荡，碰到同学不管认不认识都打个招呼，交流一下他们面试的老师，把没名额的在表格上划去。

谢晓明老师

最后，看名单上谢晓明老师还有两个名额，他的方向是超导量子器件与应用。于是就和一个同样没找到导师的同学一起去找谢老师。

谢老师是微系统所执行副所长，找他不容易。表格上标的办公室并不是他的办公地点，几经周折，终于找到他的办公室。

我们很有礼貌的跟老师问好，说明来意，奉上简历还有成绩单。谢老师打电话找了另一个老师，说他的团队都在实验室，他要招学生，一定要得到团队同意，
所以找一个实验室的老师来一起面试。

过了一会，老师没来，谢老师怕浪费我们的时间，让我们先去别的老师那边看看。

于是我们又出来游荡了一会，遇到别的同学继续问问情况，删掉一些没有名额的老师。

最后还是返回谢老师办公室，他又打了个电话，最后老师终于来了。

由于接近午饭时间，老师简单的看了我们的成绩单和简历，让我们四人一起吃饭。

饭桌上，大家就超导的相关问题做了交流。然后，谢老师吃完先走了，说去办公室等我们。

这时，实验室的王老师坦白和我们说，我们两个人里面他会更偏向于选我，因为我的数理基础更好，但是他也说，如果谢老师说不行，那也没办法！

回到办公室，两位老师坐下，开始正式面试，还是之前复试那一套，我先自我介绍，问了我一些问题。

问我导体和绝缘体还有半导体，接着问我能带的形成，还问我准粒子，我回答的不是很好。

最后谢老师说我们都不行，我量子力学固体物理什么都没学过！基础很差。这时王老师打圆场，说我们讲的比较简单。

最后，我们留下号码，谢老师说要和王老师讨论一下，会很快给我们答复。

我估计没什么戏，哎，和同伴一起出来，在外面没有希望的游荡。他去之前联系的导师那边，发现之前的导师也没名额了，哎。

在超导实验室那栋楼带了一会，电话响了，是王老师，说要见我一下。

到约定地点后，王老师坦白说谢老师对我不是很满意，但是他决定要我，他说也不管我是常州工学院什么的，只有两个要求：

  1. PhD必须读
  2. 把缺的知识补回来

还问我身体怎么样，我一下子反应过来，以后要熬夜！！
我当然说身体好了，看来以后要多加锻炼，进一步增强体质了。

后来，王老师和我说了很多，成功的人不是那些聪明的，恰恰是那些执着的人！他看我每门课成绩都还不错，觉得我比较细心，
最后他给谢老师发消息说决定要我，其实谢老师不是很看好我，王老师让我好好努力。

总的来说，能签到老师就很不错了，有人愿意要我已经是很幸运的事情了。非常感谢王老师的收留，
今后一定好好努力，走之前王老师交代了任务，回来背书，背四本书：固体物理，半导体物理学、半导体物理与器件、超导物理

这可能是我这辈子最疯狂的事情，背书，不过还是答应了，这可能是王老师一贯训练学生的做法吧。要听话！！

现在想来，好的导师，是很注重学生的基础的，他们考虑的是学生对他们今后研究是否有价值。
现在，学校基本上什么热门上什么课，学生学的多，但什么也学不到，不伦不类，这一点，学校要好好反思！
很庆幸，我选到了导师，虽然是上科大名额，与上海科技大学联陪，但是有老师收留我就很好了，但是，以后的学弟学妹，是不是还会遇到同样的问题，就很难说了！！
