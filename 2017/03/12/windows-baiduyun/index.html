其实，个人觉得，百度云超级会员（个人文件多，只好开超级会员满足空间需求。）购买3年的价格完全可以自己搭建一个VPS-私人云盘，而且安全、永久使用，不像百度云动不动和谐文件，你和谐小黄片也就算了，把我的考研视频都和谐了，不要脸，我们的资料存在上面是不安全的！

好，冷静冷静，开始进入正题！

使用教程

Aria2的配置

提供的Aria2是配置好的，傻瓜式操作，打开就能用

  1. 文件下载后如下



  2. 解压Aria2



  3. 双击aria2.exe，程序就开始运行了，不过不会有UI界面，在桌面右下角会出现程序图标，表示正在运行。




    以及



    以及


  4. 打开浏览器，输入http://aria2c.com/，即可看到程序的UI界面，很简洁



  5. 补充：程序的参数修改和使用步骤可以查看README.md文件（用浏览器打开即可）

百度云下载插件安装

  1. 解压百度云下载插件的压缩包

  2. 打开Google Chrome浏览器（没有的话安装有一个）

  3. 点击设置，然后点击页面左侧的扩展插件




    以及


  4. 勾选开发者模式



  5. 点击加载已解压的扩展程序，定位至解压的百度云下载插件的路径，并点击解压后文件夹内的chrome文件夹，点击确定。



  6. 至此，下载插件安装完成，显示如下，浏览器右上角也会出现插件图标。



具体使用

  1. 在Google Chrome中打开百度云，此时在上传一栏会出现’导出下载’，表示插件运行成功。



  2. 选中要下载的文件（可以多个，也可以是文件夹），点击’导出下载’——>’Aria2 RPC’



  3. 提示文件下载成功，此时文件会自动出现在Aria2中进行下载，浏览器访问http://aria2c.com/即可查看详细情况，速度相当快！




    以及


  4. 文件下载好后，会存放在aria2文件夹内的Aria2Data文件夹中。




    以及


一点说明

  1. 必须先运行aria2，再进行下载！

  2. 为了使用方便，可以把网址http://aria2c.com/收藏在浏览器中，随时访问。



  3. 关闭浏览器后在次打开时，会出现如下提示，我们点击取消即可。



  4. 解压后的百度云插件文件夹不要删除，否则下次启动浏览器就无法加载该插件。
