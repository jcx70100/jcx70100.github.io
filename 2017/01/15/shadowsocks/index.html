各种系统下Shadowsocks客户端的安装与配置

发表于 2017-01-15 | 更新于 2017-04-20 | 分类于 瞎折腾 | | 阅读次数

下面介绍如何在 Windows / Mac / Linux(Ubuntu) / Android 系统下对Shadowsocks的客户端（下面简称SS）的安装和配置，以便于使用相应的VPN服务。在此之前假设你已经知道了SS服务器的端口和密码，如果不知道的话，可以向VPN的提供者（管理员）索要。这里假设你使用的是我的VPN服务，端口号和密码已经获取得到，请结合自己所使用的系统环境找到对应的安装配置方法，在需要输入端口号和密码的地方输入即可。

+

Windows下SS客户端的安装与配置

适用于Windows 7及以上的系统。

第一步：下载SS客户端

下载方式：百度云

下载链接：点击此处

密码：gtzc

下载解压后，你会看到一个Shadowsocks.exe的文件。

第二步：配置客户端

运行exe文件，你会看到如下窗口；请按照图片中的内容进行配置：



输入端口号和密码，点击OK，右键右下角的运行图标：勾选[启动系统代理]，之后你应该就可以正常访问外网了。

注意，有些时候会出现网页无法访问的情况，尤其是国外网站，比如one drive，一会可以访问一会不可以访问，这是由于我们的dns被污染了，目前的解决办法是右键点击shadowsocks任务栏图标，在PAC一栏点击从GFWList更新PAC，软件会自动更新PAV列表，更新完毕后，网页即可正常访问！
