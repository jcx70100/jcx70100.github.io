刘一男带你狂撸8000词汇-公益课程（一）

发表于 2017-01-13 | 更新于 2017-04-09 | 分类于 考研 | | 阅读次数
内容摘自“刘一男”公众号

—

我爱17考生，17考生是我微信真正做起来的第一届学生，春天时的互撩，暑假你们支持我的1575直播课，秋季的感动，冬季你们收获，我的不舍，我都记得，我不会忘。

我向来不屑煽情，但我喜欢被感恩，我愿意做善事，以后好拿这些我做过的善事去吹牛B，去保持自己高洁的气场。

今天我再做一件好事。

我要免费在微信公众号开一个月多的《刘一男超越5000直达8000单词》的直播课。

+

8000词汇课很有用，研究生阶段的英语，专四专八，雅思托福，医学词汇，考博，其实就是这超越5000的8000词汇。

这个课程中，考研大纲5500，绝对不提，但是难免有时会涉及到。
如果你词汇量真得到了8000，你的词汇量在中国人中，算是很突出了。

我曾说过，5000是个坎，8000是个尖儿。
在国内，只有我，能帮你到这个尖儿。

你说男哥你免费开课，你靠什么吃饭？
我靠免费来积德，我想你们会帮我去给你们师弟师妹说我的好话的，我早晚会有收获的时候，不是么？
这个课，去年寒假是收费的，招了近1000人吧，每人200块钱，20万，我分10万，也就那么回事。
今年我不要这10万，我去年的经济学人的3万不也是捐给壹基金了么。另一方面，你知道，我不拿你钱，我讲课就没有压力，我就更放开，可以挥洒自如，汪洋恣肆，可以不考虑你是我的客户，那我就能发挥的更好，我就等于做口碑了。

一直有人问我，男哥，开不开雅思托福词汇，有没有考博，好，你考完研，你就跟男哥一起冲向单词的学术高潮吧。
有的会很黄，你做好准备。

授课方式:
1. 第一天发放课件，你把思维导图里的的单词抄在小本本上，先背，感性积累
2. 第二天，我就讲授记忆绝学，同时拓展，每天大概词汇量50左右，大概40天，两千核心词就出去了
3. 然后我课后，赠送一些资料，你再继续前行前行，走向8000。

同时，男哥单词技术天下第一，你也可以问过来，一些在5000到8000这个程度，你特别不好记的单词。我会甄选，放到课程中。我本人特别喜欢记这个领域的单词，咱们欢乐共度。
